Menu Utama

Input Perjalanan
Tambahkan data perjalanan baru berdasarkan driver, siswa, jarak, trayek, dan total biaya.
🧑‍✈️
Manajemen Driver
Tambah, lihat, ubah, dan hapus data driver untuk kebutuhan perjalanan.
✏️
Edit Perjalanan
Cari data perjalanan, ubah data, perbarui total, atau hapus data yang tersimpan.
💾
Backup & Restore
Backup data, hapus isi Firebase, lalu restore kembali dari file JSON saat dibutuhkan.
📊
Laporan Perjalanan
Lihat rekap data perjalanan, filter laporan, lalu export atau kirim ke WhatsApp.
⚙️
Konfigurasi Harga
Atur harga dasar dan kenaikan harga per kilometer agar hasil perhitungan lebih akurat.

Catatan

Pastikan file index.html, dashboard.html, input.html, driver.html, cari.html, backup.html, lapor.html, dan config.html berada dalam folder yang sama agar semua menu berjalan normal.
